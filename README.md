# 🎴 记忆翻牌游戏

一个使用纯 HTML5 / CSS3 / JavaScript 开发的经典记忆翻牌小游戏，无需任何构建工具或服务器。

---

## 🎮 游戏玩法

1. 选择难度等级（简单 / 普通 / 困难 / 限时挑战）
2. 点击任意卡片翻开它
3. 再翻开第二张卡片：
   - ✅ **图案相同**：两张卡片保持翻开状态，匹配成功
   - ❌ **图案不同**：1 秒后两张卡片自动翻回
4. 继续翻牌，直到匹配所有卡片对
5. **连击奖励**：连续匹配成功可触发 Combo 连击，飘出"Combo x2"等浮动文字，连击越高字体越大
6. **限时挑战**：在 120 秒内完成所有匹配，倒计时最后 30 秒会有红色脉动预警，时间归零则游戏失败

---

## 📋 游戏规则

| 项目 | 说明 |
|------|------|
| **翻牌次数** | 每翻开一对卡片（第二张翻开时）计为 1 次 |
| **计时器** | 从第一次翻牌开始计时，通关时停止 |
| **操作历史** | 记录最近 8 次翻牌操作及匹配结果 |
| **最佳记录** | 每个难度单独保存本地最佳成绩（限时挑战除外） |

## 🏆 评级标准

通关后根据表现获得星级评价：

### 普通模式（简单 / 普通 / 困难）

根据翻牌次数评定：
- ⭐⭐⭐ **三星**：翻牌次数 ≤ 卡片对数 × 1.5
- ⭐⭐ **二星**：翻牌次数 ≤ 卡片对数 × 2.5
- ⭐ **一星**：翻牌次数 > 卡片对数 × 2.5

### ⏳ 限时挑战模式

根据通关时的**剩余时间**评定：
- ⭐⭐⭐ **三星**：剩余时间 ≥ 60 秒
- ⭐⭐ **二星**：剩余时间 ≥ 30 秒
- ⭐ **一星**：剩余时间 < 30 秒

> 限时挑战的最佳记录以「剩余时间最多」为核心指标，星级相同则比较剩余时间，再比较翻牌次数。

## 🔊 音效系统

游戏内置纯代码合成的音效系统，**无需任何外部音频文件**，全部通过 Web Audio API 的振荡器实时生成：

| 音效 | 触发时机 | 合成方式 |
|------|----------|----------|
| 翻牌声 | 点击翻开卡片时 | 双音正弦波（800Hz → 1000Hz） |
| 匹配成功声 | 两张卡片图案一致时 | 三音上升和弦（C5 → E5 → G5） |
| 匹配失败声 | 两张卡片图案不同时 | 双音下降锯齿波（300Hz → 250Hz） |
| 通关声 | 全部匹配完成时 | 六音阶上升旋律 |
| 失败声 | 限时挑战超时的时候 | 三音下降锯齿波 |

点击游戏界面上的 🔊 / 🔇 按钮可随时切换音效开关状态。

## 🔥 连击奖励（Combo）

连续匹配成功时会触发连击奖励机制：

- 连续匹配成功 2 次及以上，在匹配卡片位置飘出"Combo xN"浮动文字
- 连击数越高，浮动文字字号越大（1.2rem + 0.3rem × 连击数，上限 3.5rem）
- 匹配失败时连击数清零
- 顶部状态栏实时显示当前连击数（🔥 连击）
