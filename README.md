# 🎴 记忆翻牌游戏

一个使用纯 HTML5 / CSS3 / JavaScript 开发的经典记忆翻牌小游戏，无需任何构建工具或服务器。

---

## 🎮 游戏玩法

1. 选择难度等级（简单 / 普通 / 困难 / 限时挑战）
2. 点击任意卡片翻开它
3. 再翻开第二张卡片：
   - ✅ **图案相同**：两张卡片保持翻开状态，匹配成功
   - ❌ **图案不同**：1 秒后两张卡片自动翻回
4. 继续翻牌，直到匹配所有卡片对
5. **连击奖励**：连续匹配成功可触发 Combo 连击，飘出"Combo x2"等浮动文字，连击越高字体越大
6. **限时挑战**：在 120 秒内完成所有匹配，倒计时最后 30 秒会有红色脉动预警，时间归零则游戏失败

---

## 📋 游戏规则

| 项目 | 说明 |
|------|------|
| **翻牌次数** | 每翻开一对卡片（第二张翻开时）计为 1 次 |
| **计时器** | 从第一次翻牌开始计时，通关时停止 |
| **操作历史** | 记录最近 8 次翻牌操作及匹配结果 |
| **最佳记录** | 每个难度单独保存本地最佳成绩 |

---

## ⭐ 难度等级

| 难度 | 网格 | 卡片对数 |
|------|------|----------|
| 🟢 简单 | 3 × 4 | 6 对 |
| 🟡 普通 | 4 × 4 | 8 对 |
| 🔴 困难 | 5 × 4 | 10 对 |

---

## 🏆 评级标准

通关后根据表现获得星级评价：

- ⭐⭐⭐ **三星**：翻牌次数 ≤ 卡片对数 × 1.5
- ⭐⭐ **二星**：翻牌次数 ≤ 卡片对数 × 2.5
- ⭐ **一星**：翻牌次数 ≤ 卡片对数 × 4

## 🚀 运行方式

直接双击 `index.html` 即可在浏览器中打开游戏，**无需任何依赖**。

或者使用本地服务器（推荐）：

```bash
# Python 3
python -m http.server 8000

# 然后访问 http://localhost:8000
```
